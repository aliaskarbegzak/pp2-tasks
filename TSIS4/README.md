# TSIS4 Snake

Run:

```bash
python main.py
```

Install:

```bash
pip install pygame psycopg2-binary
```

Create PostgreSQL database:

```sql
CREATE DATABASE snake_tsis4;
```

Then edit `config.py` and write your PostgreSQL password.

Tables are created automatically when `main.py` starts.
You can also run `schema.sql` manually in pgAdmin.
