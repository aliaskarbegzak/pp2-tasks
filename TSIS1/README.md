# TSIS1 PhoneBook

## Run

```bash
python phonebook.py
```

## Install

```bash
pip install psycopg2-binary
```

## Database

Create database:

```sql
CREATE DATABASE phonebook_tsis1;
```

Then edit `config.py` and write your PostgreSQL password.

Inside app choose:

```text
1. Create schema
```

This creates tables, procedures and functions.
